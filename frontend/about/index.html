About SketchMyInfra

SketchMyInfra is a free AI architecture diagram generator that turns plain English infrastructure descriptions into production-ready cloud diagrams — no drag-and-drop, no templates, no design skills needed.

What is SketchMyInfra?

SketchMyInfra is an AI-powered tool that converts natural language into architecture diagrams for AWS, GCP, and Azure. You describe your infrastructure — like "a three-tier AWS app with ALB, ECS Fargate, and RDS PostgreSQL in private subnets" — and it generates a structured, labeled diagram in seconds.

Under the hood, it uses Google Gemini AI to understand your description and generate PlantUML code, which is then rendered into an SVG or PNG diagram using the official PlantUML server. The PlantUML code is also shown so you can modify and version-control it.

It's built for DevOps engineers, solution architects, and developers who need to document or communicate infrastructure designs quickly — without spending hours in a drag-and-drop tool.

Why we built it

Architecture diagrams are essential — for onboarding, incident reviews, architecture reviews, and documentation. But creating them is slow and tedious. Existing tools require you to drag, drop, align, and connect boxes manually. Templates help a little, but you still spend more time on the tool than on the actual thinking.

SketchMyInfra flips that. You focus on describing what you want to build. The AI handles the diagram. In most cases, you have a usable diagram in under 30 seconds — and you can export the PlantUML code to refine it further in any editor.

How it works — technically

  1. 1

    You type a description

    e.g. "AWS three-tier app with CloudFront, ALB, ECS, and Aurora PostgreSQL"

  2. 2

    Gemini AI generates PlantUML

    The backend sends your description to Google Gemini with cloud-specific instructions and icon mappings.

  3. 3

    PlantUML renders the diagram

    The generated code is sent to a self-hosted PlantUML server which returns SVG/PNG.

  4. 4

    You download or iterate

    Export as SVG/PNG, copy the PlantUML source, or refine your description and regenerate.
