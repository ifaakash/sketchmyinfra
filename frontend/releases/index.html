Release Notes

What's new in SketchMyInfra. Every feature, improvement, and fix — documented.

Jun 1, 2026 Bug Fix

International Language Support (CJK Fix)

Fixed rendering failures for diagrams with Chinese, Japanese, Korean, and other non-Latin labels. Two issues were at play: the PlantUML server wasn't told the request body was UTF-8, so multi-byte characters were garbled into syntax errors. Additionally, CJK characters consume 2–3x more tokens than English text, causing the AI to hit output limits and return truncated diagrams mid-line.

  • Added charset=utf-8 to PlantUML server requests — CJK, Hindi, Arabic, Cyrillic labels now render correctly
  • Doubled AI output token limit (4K → 8K) to handle token-heavy non-Latin scripts
  • Truncation detection — clear error message instead of cryptic PlantUML failures when output is cut off
May 31, 2026 Bug Fix

Hardened PlantUML Rendering Pipeline

Fixed a class of diagram rendering failures caused by special characters in labels. The AI sometimes generated characters like &, ~, or smart quotes that PlantUML interprets as syntax operators. Added a comprehensive pre-processing layer that automatically sanitizes these before rendering.

  • Auto-sanitizes & ~ | and HTML entities inside diagram labels
  • Strips trailing semicolons, smart quotes, zero-width characters, and BOM markers
  • Updated AI system prompt to prevent generating problematic characters
May 29, 2026 New Feature

Admin Dashboard & Generation Analytics

Track diagram generation health with a new admin panel showing global statistics, failure rates, and recent error details. Per-user stats are also available for authenticated users.

  • Generation status tracking — success, failed, and pending states
  • Admin dashboard with global analytics and recent failure details
  • Per-user stats API for authenticated users
May 15, 2026 New Feature

Save & Share Drawings

Your freehand drawings now persist across sessions. Sign in to save to the cloud and share with anyone via link. Anonymous users get local browser storage.

  • Auto-save with 3-second debounce — never lose work
  • Drawings gallery with create, rename, and delete
  • Share any drawing via public link — recipients can view and fork
  • One-click migration from local storage to cloud on sign-in
May 10, 2026 New Feature

Freehand Drawing — Whiteboard Mode

You can now sketch infrastructure diagrams by hand using our new Draw tool — a full-featured whiteboard powered by Excalidraw. Shapes, text, arrows, connectors, and freehand drawing — all free, right in your browser.

  • Full whiteboard with shapes, text, arrows, and freehand drawing
  • Export to PNG and SVG directly from the canvas
  • Dark and light theme toggle
  • No signup, no limits — completely free to use
